# Lucas-English-C-Sharp-Examples
This repository contains scipts I've made in C# for practice and optimization of my stealth-based dungeon heist game, DEPTHSEEKER.

## Contents
- **TimerManager.cs**: Cleanly manages timers for various gameplay systems.
- **BuffManager.cs**: Handles player buffs and related UI interactions.
- **FlySpawn.cs**: A firefly like creature that navigates and can be instanced many times.
- **AiController.cs**: Manages AI behaviors using a utility-based system.
